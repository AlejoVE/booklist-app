## Introduction:

This project consists of an app that displays a list of books stored by the user. It was created following the [Traversy Media tutorial](https://www.youtube.com/watch?v=JaMCxVWtW58)

## Technologies:
- HTML5
- CSS
- Javascript
- GitKraken
- Font Awesome
- Bootswatch - Yeti theme
